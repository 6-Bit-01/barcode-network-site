# Database Images

Drop dossier images here. Reference them in `src/content.ts` like:

```
image: "/database/6bit.png",
```

**Recommended specs:**
- Square aspect ratio (1:1)
- Minimum 400×400px
- PNG or WebP preferred
- Dark/moody aesthetic to match the site

**Special images you mentioned:**
- A default placeholder for entries without images
- A "CLASSIFIED" / confidential image for restricted entries

Name files to match the entry slug (e.g. `6-bit.png`, `barcode-radio.png`).
